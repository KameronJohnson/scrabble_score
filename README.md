# scrabble_score
User inputs a word and scrabble_score() displays the Scrabble score of that word.

## Setup Instructions

###Gems used
```erb
sinatra
sinatra-contrib
```

###Install Bundler
```erb
$ gem install bundler
```

###Run Bundler
```erb
$ bundle
```
